# Known Limitations

- Scoring does not verify correctness.
- Current profile is biased toward electronics/amplifier threads.
- Loudspeaker/crossover terms are not fully covered yet.
- Some useful short posts may score too low.
- Some verbose technical-looking posts may score too high.
- Attachments/images are detected but not analyzed.
- The tool should be used on one thread at a time.



Linux binary support:
- Current build environment only
- glibc 2.38 or newer required
- Older Linux/macOS builds are out of scope for beta
